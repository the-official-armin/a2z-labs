# Anime Face Generator (DCGAN)

A Deep Convolutional Generative Adversarial Network (DCGAN) built with TensorFlow and Keras that generates 64×64 anime-style faces. This project demonstrates the complete GAN pipeline, including dataset preprocessing, generator and discriminator architectures, training loop, and image generation.

> **Note:** Due to hardware and time limitations, the model was not trained to convergence. Training a DCGAN on a CPU can take several hours depending on the dataset size and hardware.

---

## Features

- Loads images directly from a dataset directory
- Preprocesses images to the range `[-1, 1]`
- DCGAN Generator using transposed convolutions
- DCGAN Discriminator using convolutional layers
- Custom GAN training loop
- Generates and displays anime faces after training

## Model Architecture

### Generator

The generator accepts a 100-dimensional random noise vector and progressively upsamples it into a 64×64 RGB image using:

- Dense layer
- Reshape
- Conv2DTranspose layers
- Batch Normalization
- ReLU activations
- Tanh output activation

**Input:** `(100,)`

**Output:** `(64, 64, 3)`

---

### Discriminator

The discriminator classifies images as real or fake using:

- Conv2D layers
- LeakyReLU activations
- Batch Normalization
- Flatten layer
- Dense sigmoid output

**Input:** `(64, 64, 3)`

**Output:** Probability that the image is real.

## Training

The training loop alternates between:

1. Training the discriminator on:
   - Real images
   - Fake images generated by the generator

2. Training the generator to fool the discriminator.

Binary Cross Entropy loss is used for both networks with the Adam optimizer.

---

## Image Generation

After training, the generator creates images from random noise vectors.

Generated images are:

1. Converted from `[-1,1]` back to `[0,1]`
2. Clipped to valid pixel values
3. Displayed in a 4×4 image grid

## Current Results

The project successfully:

- Builds both neural networks
- Compiles the GAN
- Performs training steps
- Generates sample anime faces

However, the model was **not fully trained**.

Training a DCGAN to produce high-quality images typically requires:

- Thousands of training iterations
- Multiple hours of training (especially on a CPU)
- A dedicated GPU for practical training times

Because of limited time and hardware, this repository demonstrates the implementation rather than a fully converged model.
